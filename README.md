# TnP Portal

A full-stack Training & Placement (TnP) portal for colleges to manage job postings and student applications in a structured, scalable way.

This project is being built incrementally with a strong backend foundation and clean database design.

---

##  Core Goals

- Allow **TnP Admins** to post job opportunities
- Allow **Students** to view eligible jobs and apply
- Track applications with clear status
- Maintain clean separation of roles and data
- Keep the system simple, secure, and extensible
